L’algorithme des k plus proches voisins¶

Introduction - principe¶

L’algorithme des \(k\) plus proches voisins (\(k\)nn en anglais pour k nearest neighbors) appartient à la famille des algorithmes d’apprentissage automatique (en anglais machine learning). L’apprentissage automatique (ou machine learning) peut être vu comme l’extraction de connaissances à partir de données. Les champs d’application sont très vastes (recommandations ou publicités ciblés, identification de caractères manuscrits, détection d’activités frauduleuses dans des transactions bancaires, lutte anti-spam etc.)

L’algorithme a besoin d’un ensemble de données \(E\). Chacune de ces données est caractérisée par sa catégorie, on dit aussi sa classe. Il s’agit d’associer une donnée nouvelle \(u\), n’appartenant pas à \(E\) à une catégorie en se basant sur la catégorie majoritaire chez ses \(k\) plus proches voisins, \(k\) étant un entier positif. Décrit de cette façon, il s’agit ici d’un problème de classification. Pour réaliser cet objectif, on a besoin d’une fonction qui donne la distance entre deux données.

Principe

  • On calcule les distances entre la donnée \(u\) et chaque donnée appartenant à \(E\) à l’aide de la fonction distance;

  • On retient les \(k\) données du jeu de données \(E\) les plus proches de \(u\);

  • On attribue à \(u\) la catégorie majoritaire chez les \(k\) données les plus proches.

Calcul de distance¶

On utilisera ici à la distance euclidienne. Rappel: soient deux points \(A(x_1, x_2)\) et \(B(x_1^{'}, x_2^{'})\) du plan, la distance \(d(A,B)\) est:

\[d(A,B)=\sqrt{(x_1-x_1^{'})^2+(x_2-x_2^{'})^2}\]

Dans l’implémentation en python ci-après, les coordonnées sont transmises à la fonction dist dans un tableau.

Remarque

On peut généraliser la définition de la distance \(d(A,B)\) dans le plan à un espace de dimension \(n\). Soient par exemple \(A(x_1, x_2, \ldots, x_n\)) et \(B(x^{'}_1, x^{'}_2, \ldots, x^{'}_n)\), on a alors:

\[d(A,B)=\sqrt{(x_1-x_1^{'})^2+(x_2-x_2^{'})^2+ \ldots + (x_n-x_n^{'})^2}\]
import math


def dist(coordA, coordB):
    """
    Retourne un flottant positif correspondant à la distance entre A et B;
    coordA, coordB : tableaux de flottants correspondant aux coordonnées de ces points
    """
    s = 0.0
    for i in range(len(coordA)):
        s += (coordA[i] - coordB[i]) ** 2
    return math.sqrt(s)

def knn(data, X, k):
    """
    Renvoie un tableau contenant les éléments du jeu de données data associés avec leur distance avec X. 
    Ce tableau ne contiendra que les k premiers éléments plus proches de X.
    data: tableau de dictionnaires contenant les caractéristiques des données;
    X: tableau de flottants, correspondant aux coordonnées de l'élément inconnu X dont 
    on cherche la catégorie.
    k: entier positif (correspond au nombre de voisins à considérer)
    """
    
    tdist = [None] * len(data)#création d'un tableau qui stockera les caractéristiques des iris ainsi
    #que leur distance avec X, dans un dictionnaire.
    j = 0#un compteur
    for iris in data:
        c_iris = [iris['petal_length'], iris['petal_width']]
        tdist[j] = {'petal_length': iris['petal_length'],
                   'petal_width': iris['petal_width'],
                   'species': iris['species'],
                   'dist_X': dist(c_iris, X)}
        j = j + 1
    #fonction qui retourne la valeur de la clé 'dist_X' d'un dictionnaire
    def plusproche(x):
        return x['dist_X']
    
    tdist.sort(key=plusproche)
    return tdist[:k]
